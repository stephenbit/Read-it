# read://it

Latest build: https://read-it-news-aggregator-app.herokuapp.com/

Navigate to /client in terminal then:

`npm i`

Open a new tab and navigate to /client in terminal then:

`npm i`

In the client tab:

`npm run serve`

In the server tab:

`npm run server:dev`

Go to the following address in your web browser:

http://localhost:8080/
